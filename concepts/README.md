# concepts/

Parking lot for standalone app ideas that would consume Kvalt. Each idea gets its own folder with a `README.md` covering the pitch, the reference use case, the architecture sketch, and the open questions.

When an idea is ready to become a real project, it graduates out of `concepts/` into its own repo (or sibling package, once Kvalt becomes a monorepo).

## Current concepts

- [`typewriter-app/`](./typewriter-app/) — Beautiful typewriter-soul creative writing canvas with Claude. Floating knowledge folders cluster by recency and topic relevance.
