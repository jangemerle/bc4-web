## Key Takeaways
### 1. Five Marketing Automation Playbooks
(1) Generate Instagram Reels in brand style from a single prompt. (2) Scrape competitor trending Reels → analyze patterns → generate new scripts. (3) Find trending podcasts in niche → create social posts from content. (4) Generate 100 LinkedIn outreach targets with personalized openers. (5) Analyze competitor Facebook/Google ads (duration, engagement) → generate new concepts.

### 2. API Stack: Apify + Kai
Never ask Claude Code to scrape directly. Use Apify API for social media and ads data. Use Kai AI for video and image generation. Two APIs, endless data sources.

### 3. Reference Images for Brand Consistency
Feed Instagram screenshots into Claude Code as brand style context. The model learns from visual examples and reproduces that aesthetic across generated content.

### 4. Voice Input via Whisper Flow
Dictate campaign briefs directly to Claude Code using voice. Faster ideation, more natural instructions, hands-free operation.

### 5. Beautiful HTML Output
Deliverables aren't CSV or JSON — they're styled HTML pages matching your brand. Ready to share with stakeholders, no formatting needed.

### 6. 10-Minute Setup, No Coding
The entire workflow requires less than 10 minutes to configure. No JavaScript, Python, or backend knowledge. Prompt + API key + reference images = done.
