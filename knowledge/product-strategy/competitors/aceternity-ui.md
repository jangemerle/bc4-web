# Aceternity UI

**URL:** https://ui.aceternity.com
**Category:** Competitor (closest comparable — animation-focused component library)
**Added:** 2026-03-25

---

## What it is
330+ animated React components (200+ free, 100+ pro). Copy-paste model, also available via shadcn CLI 3.0 and MCP server. Founded by Manu Arora, solo creator. $80K/month within 2 months of Pro launch. 120,000+ users.

## Pricing
- Free: 200+ components, full copy-paste access
- Pro: $199 one-time lifetime — 100+ premium blocks + 30+ templates

## Technical approach
- React + Next.js + Tailwind CSS v4 + **Framer Motion** (not custom motion primitives)
- Copy-paste primary, shadcn CLI secondary, MCP server emerging
- Components curate and package Framer Motion — no custom animation layer
- Landing page / marketing site focused (not app components)

## Distribution (the model to study)
1. Copy-paste from website (primary)
2. shadcn CLI 3.0 registry: `npx shadcn@latest add @aceternity/3d-marquee`
3. MCP server for AI assistant integration
4. Twitter + ProductHunt for acquisition

## Key strengths
- Animation quality (curated Framer Motion)
- Breadth (330+ components, 30+ templates)
- One-time pricing (low friction)
- Active founder with community presence
- Daily release cadence signals momentum
- Free tier is genuinely generous

## Key weaknesses (= Kvalt's moat)
- **No design token system** — components hardcode values
- **No unified personality layer** — can't switch the overall feel
- **Landing page focused** — limited app/dashboard components
- **Framer Motion dependency** — adds bundle size, not custom
- **One aesthetic** — no "faces" concept, no personality switching
- **Limited accessibility depth**
- **Smaller community** (3.3K GitHub stars vs shadcn's 109K)

## Relevance to Kvalt
Closest comparable product and strongest proof of market. Validates: solo founder can succeed, animation niche pays, one-time pricing works, generous free tier + dramatically better Pro is the model. But Aceternity sells animated components. Kvalt sells design personalities. Complementary, not directly competing — a developer could use Aceternity components styled with a Kvalt face.
