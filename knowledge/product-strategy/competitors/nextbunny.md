# NextBunny

**URL:** https://nextbunny.co
**Category:** Competitor (adjacent — visual builder, potential distribution channel)
**Added:** 2026-03-25

---

## What it is
Free visual drag-and-drop builder for production-ready React/Next.js apps. 100+ templates. Generates clean TypeScript + Tailwind CSS v4 + semantic HTML. No proprietary runtime. shadcn/ui compatible. AI code generation in development.

## Positioning
"The only free drag-and-drop builder for production-ready Next.js projects." Bridges design and code. Code quality emphasis — "generates the same clean code you'd write by hand."

## Pricing
Free (early access). Business model not yet defined.

## Relevance to Kvalt
1. **Potential distribution channel** — if NextBunny supports design system presets/themes, Kvalt faces could be selectable within the builder
2. **Validates the visual builder market** — people want to build visually but with real code output
3. **Not a direct competitor** — NextBunny is a builder tool, Kvalt is a design personality layer. Complementary if integrated.
