## What matters for Kvalt

### Figma Console MCP (open source, MIT)
89+ tool MCP server bridging Figma ↔ AI assistants. Two-way communication — can read AND write to Figma. Described as "totally blows Figma's official tooling out of the water."

Key capabilities:
- Design token & variable extraction and creation
- Component building directly in Figma via AI
- Screenshot capture for visual context
- Real-time plugin debugging
- FigJam support
- Cloud write relay for web AI clients

4 connection modes: NPX (full, 89+ tools), Cloud (79 tools), Remote SSE (22 read-only), Local Git (contributors).
