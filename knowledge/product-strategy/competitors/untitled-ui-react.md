# Untitled UI React

**URL:** https://www.untitledui.com/react
**Category:** Competitor (adjacent — component library, not design personality)
**Added:** 2026-03-25

---

## What it is
Massive React component library — 5,000+ component variants, 420+ page examples, React 19 + Tailwind v4 + React Aria (Adobe). Copy-paste architecture (similar to shadcn). Backed by the most popular Figma UI kit on the market.

## Pricing
- Free: 2,000+ components, 20 page examples, 900+ icons
- Pro Solo: $129 one-time (full kit + Figma variables + dark mode + 420+ page examples)
- Team plans available

## Key strengths
- Figma ↔ code synchronization (every component mirrored)
- React Aria for accessibility (vs. Radix which shadcn uses)
- Enormous breadth — covers nearly every UI pattern
- One-time purchase, no subscription

## Key weaknesses (= Kvalt's opportunity)
- **One aesthetic** — no personality switching, no faces concept
- **No motion system** — components are functional, not animated
- **No opinionated defaults** — gives you options, not opinions
- Doesn't solve the "all my projects look the same" problem

## Relevance to Kvalt
Untitled UI validates the market for paid UI components ($129-299 range). But it competes on breadth, not personality. Kvalt's "faces" concept is orthogonal — Kvalt could theoretically sit on top of Untitled UI the same way it sits on top of shadcn. Not a direct competitor; more of a "same shelf, different aisle" product.
