## The Model

Build Google Sheets templates (budgets, trackers, planners, databases) and sell them on Etsy. Economics are compelling: digital products, delivered instantly, infinite resale, 90-95% margins.

**Revenue examples:** Budget planners generate $3,700–$19,458/month depending on niche, marketing, and bundle strategy. These are not unicorn exceptions — they're repeatable middle-class incomes from a single template that was built once.

## The Workflow

Claude Code + Google Workspace CLI automates the entire generation pipeline. Instead of building each template manually, describe the template spec and let Claude Code generate the sheet structure, formulas, styling, and interactivity. Google Workspace CLI handles creation, template publishing, and variant generation.

One advantage over pure code products: sheets are inspectable by customers. They can see exactly what they're buying — no black box. This builds trust and justifies the price.

## Why Bundles Win

Selling bundles (Budget + Savings + Debt Tracker as a set) outperforms single templates. Customers perceive more value, and the marginal cost to create a bundle (given you already built the first template) is minimal. A bundle at $49 sells better than trying to sell three separate $19 templates.

## Why This Matters

This validates a design system utility: templating at scale. Kvalt could theoretically follow this pattern — ship design system "bundles" (basic landing page kit, SaaS dashboard kit, ecommerce page set) as downloadable Figma files or HTML/Tailwind templates. Build once, sell many. The economic model is proven; it's a question of packaging and positioning.

## Relevance to Kvalt

Kvalt's long-term monetization could include a template marketplace. Design systems are a solved problem; the next layer is "design system *applications*" — pre-made screens and page templates that use Kvalt's system. Just as budget templates are more valuable than a spreadsheet template engine, pre-made dashboards using Kvalt would be more valuable than the system alone. The Google Sheets precedent shows this market is real.
