# AI Workflow Systems Market Analysis — April 2026

**Source:** Deep research across GitHub, Gumroad, Product Hunt, HN, YouTube, market data | **Added:** 2026-04-07

## Market snapshot

Selling AI project folders (.cursorrules, CLAUDE.md, skills, agent configs) is a real but rapidly evolving market in year ~1 of its "serious money" phase.

### Landscape by platform

**Cursor rules:** Massive free layer (awesome-cursorrules: 39K stars). Paid packs $5-$39 on Gumroad. Directory sites (dotcursorrules.com, cursorrules.org) are free/ad-supported.

**Claude Code skills:** 6+ marketplaces launched in 6 months — SkillsMP (700K+ skills), ClaudeSkills.ai (90% creator revenue), Agent37 (hosted access, 80% creator share), ClaudeSkillsHQ (paid tiers). Anthropic ships 16 official skills (frontend-design: 277K+ installs). 60K+ repos use AGENTS.md.

**Cross-platform:** steipete/agent-rules (5.6K stars), agent-skill-creator (673 stars, supports 14+ tools).

### Revenue signals

- Solo sellers: $500-$5K/month typical for differentiated products
- Travis Hudson: claims $312/day (~$9.4K/month) from 17 Claude skills
- Gumroad software/dev category: $65.8M total tracked, avg $60K/product
- Price clustering: $5-15 (individual packs), $19-39 (bundles), $49-150 (course combos)
- Cursor at $2B ARR, Claude Code trajectory toward $1B — the underlying platforms are massive

### Lifecycle position

Following ThemeForest pattern — suggests 3-5 year window before maturation.
- **WordPress themes:** Peak 2016, now 50-70% sales decline. 8-year cycle.
- **Figma UI kits:** Stagnating 2025 — platform absorbed features (variables, auto-layout)
- **Notion templates:** ~4 years in, approaching maturity. Domain-specific still sells, generic saturated.

Currently: past "what is this?", entering "everyone's selling one."

## Platform risk analysis

### What first-party providers are already doing
- Anthropic: 16 official skills, official marketplace architecture, SKILL.md standard they control, subsidizing Claude Code at ~25:1 cost ratio for platform lock-in
- Cursor: Rules are first-party (.mdc format), building toward Composer 2 agentic features
- OpenAI: Codex CLI adopted SKILL.md standard, building competing ecosystem

### What gets eaten (short shelf life)
- Generic framework rules (React, Python best practices)
- Basic CLAUDE.md boilerplate
- Simple prompt templates
- Setup guides that restate official docs

### What survives (durable)
1. **Domain expertise as skills** — bioinformatics, security, design systems, legal. First-party won't build niche verticals.
2. **Opinionated taste + curation** — 200+ decisions about motion, tokens, a11y, voice. Not boilerplate.
3. **Hosted access / SaaS** — sell outcomes not files (Agent37 model)
4. **Multi-agent orchestration** — Designer→Builder→Auditor pipelines too complex/opinionated for first-party
5. **Continuously maintained systems** — living workflow > static download

## Relevance to Kvalt

Kvalt's workflow system is NOT a generic rule pack — it's encoded design taste + domain expertise + multi-agent orchestration. This places it firmly in the "durable" category.

**Viable plays:**
1. Extract design system skills as standalone products (hosted, not raw files)
2. Sell the workflow as part of the full Kvalt offering (characters + skills + pipeline)
3. Open-source the generic parts (visibility), charge for the opinionated/maintained system

**Key risk:** Don't sell $9 rule packs — that's the ThemeForest long tail death zone. Sell the integrated system where the skills are one layer of a larger product (DS + themes + workflow).

**The cooking show principle applies here too:** showing the CLAUDE.md publicly sells the depth. The value isn't the file, it's the 200 decisions behind it + ongoing maintenance.
