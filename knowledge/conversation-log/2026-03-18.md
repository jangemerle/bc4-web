## Skills Built

### /check-tokens
Audits any Figma frame or React component against Kvalt's token system. Catches hardcoded values, wrong semantic tokens, off-grid spacing, radius violations. Outputs a violation table with correct token suggestions.

### /grill-component
Design tree interview for components. Walks down every branch (variants, states, content model, composition, accessibility, motion, edge cases) before writing any code. Inspired by Matt Pocock's /grill-me and Frederick Brooks.

### /tdd-component
Test-driven component development. Red-green-refactor loop: write one failing test, implement to pass, refactor, repeat. Follows confirmed design from grill-component. Ends with check-tokens audit.

### Established workflow
`grill-component → write spec → tdd-component → check-tokens (final audit)`
