## Tags that feel on-brand for Kvalt (hypothesis — not yet tested)

These feel consistent with Kvalt's personality based on voice learnings:
- `clean`, `crisp`, `whitespace` — Kvalt values negative space
- `monospace`, `code-native`, `terminal` — technical precision, dry humour
- `typographic`, `editorial` — but only when paired with a strong concept
- `geometric`, `minimal` — restraint as a statement
- `print`, `paper` — analog formats work well
- `quiet`, `subtle` — not loud, not in-your-face

## Tags to use with caution for Kvalt

- `luxury`, `high-end`, `premium` — risks feeling precious
- `bold`, `expressive`, `vibrant` — can clash with Kvalt's dry restraint
- `playful`, `friendly`, `approachable` — risks feeling generic/startup-y
- `dark-mode` — works for wild card directions, not for safe versions

---

## Tag selection process

1. Identify the **concept frame** for the version (receipt, letter, etc.)
2. Pick 2 tags that match the format (e.g., `monospace`, `paper`)
3. Pick 1–2 tags that set the mood (e.g., `quiet`, `crisp`)
4. Avoid combining tags from the "use with caution" list with each other
