# Concept Library

Real-world formats and metaphors that can frame Kvalt page designs.
This list grows after every session. Check it before writing a concept brief.

---

## How to use this

1. Look up the **page type** you're designing
2. Check which concepts are proven, promising, rejected, or untested for that type
3. Never re-use a proven concept on a second page without a strong reason — "each page needs its OWN conceptual frame"
4. Prioritise untested concepts for wild card slots

---

## Status key

- `✅ Proven` — Jan loved it. Use the format for inspiration, but don't copy it verbatim on another page.
- `👍 Promising` — Jan liked the direction. Worth exploring further.
- `❌ Rejected` — Tested and failed. Note the reason so you don't repeat the mistake.
- `🔲 Untested` — Hypothesis only. Good candidate for a wild card.

## Rejected strategies (don't go here)

These patterns have been tested and consistently fail for Kvalt:

- **"Official document" as a category** — ticket, citation, classified, memo. They're all the same authority + mundane joke. Individual formats CAN work (receipt is technically an official document) but the joke can't be "isn't it funny that this is an official document."
- **Repeating a winning format** — one receipt is clever, two is a pattern, three is a crutch. Each page needs its own frame.
- **Complex concept + complex execution** — pick one. Simple concept → rich execution, or complex concept → minimal execution.
- **Self-aware irony** — "KVALT PERMANENT COLLECTION" style. The cleverness should be invisible.
